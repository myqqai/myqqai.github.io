Hexo 文章产出流程，以及如何写出优雅高质的文章！！！

目标

通过规范的流程，输出高质量的文章。

流程设计

  • 新建草稿
  • 设计大纲
    • 简述
    • 目标
    • 正文
    • 常见问题
    • 总结
  • 检查审阅
    • 修订
    • 朋友试读
  • 发布

新建草稿

post 是默认的 layout ，可通过编辑 _config.yml 文件 default_layout 字段修改。

布局 路径
post source/_posts
page source
draft source/_drafts

为迎合文章产出流程，将默认修改为 draft

发布

书写规范

  • 简述（概述本章中心思想）
  • 代码，严格使用代码格式规范，必要注释不可缺失
  • 英文，英文与汉子间空一格
  • 关键词，粗体标记显示
  • 标点，中英文标点不可混用
